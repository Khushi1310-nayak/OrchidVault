# OrchidVault

OrchidVault is a personal focus sanctuary designed for students, developers, and creators who want to organize their work, manage resources, and stay in flow.

Built with a calm and productivity-oriented design philosophy, OrchidVault combines structured study management, music integration, workspace organization, and distraction-free workflows into a unified digital environment.

Live Site: https://orchidvault-52844219986.asia-southeast1.run.app/

---

## About

OrchidVault is designed as a personal productivity ecosystem where users can:
- organize study materials
- manage PDFs and folders
- listen to music while working
- maintain workspace cleanliness
- stay focused during deep work sessions

The platform focuses on simplicity, organization, and flow-state productivity.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Khushi1310-nayak/OrchidVault.git
cd OrchidVault
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file and add:

```env
# FIREBASE CONFIGURATION

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```

### Run Development Server

```bash
npm run dev
```
